## Abstract

So what's the difference? Usually we use these words as synonyms.
Nevertheless there's a huge difference in the way,
that you create code.

I believe that studies will write you code, that machine understands.
But then the real journey begins - to write code, that other people
can read.

Together we'll see what is the big difference in writing code
and delivering value (with writing code as a small part of the process).
