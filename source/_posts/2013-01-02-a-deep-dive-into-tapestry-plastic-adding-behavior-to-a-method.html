This we may build using static builder methods in it's class, e.g.[gist id="4432985"] Next there's a place for a delegate (meaning class transforming delegation). So now we end up needing a transformer, inside of which we'll use an advice (org.apache.tapestry5.plastic.MethodAdvice). See creating and putting them together in the following snippet:[gist id="4433014"] As you have seen by now, Plastic is a pretty high level, object oriented and well designed tool for modifying classes.
