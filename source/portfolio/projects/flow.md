#### Overview

For some time I was the only developer working on this OpenSource library for building mobile applications.
I was refactoring and reworking complex build system, based on Gradle.

Besides build capabilities Apphance Flow may deploy applications by generating and sending direct installation
QR codes to the stakeholders.
