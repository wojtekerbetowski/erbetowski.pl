#### Overview

I was involved in building a proxy between generic API, developed for the sake of JavaScript client,
and mobile application client. I had a chance to understand many mobile systems limitations
and learn to optimize API to provide possibly best user experience.

In the meantime I was asked to help introducing Agile methodology to the existing team
and that resulted in a great Kanban implementation.
